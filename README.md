# Portfolio - Antonio Benedetto

Portfolio personale di Junior Developer - Sito statico costruito con HTML, CSS e JavaScript vanilla.

## 🚀 Tecnologie

- **HTML5** - Markup semantico
- **CSS3** - Stili personalizzati con CSS Variables, Flexbox, Grid, Animazioni
- **JavaScript** - Vanilla JS per le interazioni
- **Font** - Google Fonts (Syne, JetBrains Mono)

## 📂 Struttura del Progetto

```
├── index.html        # File HTML principale
├── css/
│   └── style.css    # Foglio di stile principale
├── js/
│   └── main.js      # Interazioni JavaScript
├── images/
│   ├── logo.png     # Logo del sito e favicon
│   ├── movie.PNG    # Anteprima progetto
│   ├── plastic.PNG  # Anteprima progetto
│   └── gae.PNG      # Anteprima progetto
├── README.md        # Questo file
└── .gitignore       # Regole Git ignore
```

## 🛠️ Sviluppo

Per eseguire in locale:

1. Apri `index.html` nel browser
2. Oppure usa un'estensione live server in VS Code
